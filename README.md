# What's 4zy0
4zy0 is the kawaii language

## Features of 4zy0
- kawaii
- can't read
- inspired by Brainf*ck

## How to programing
4zy0's instructions are only 8.

|instructions|operation|
|:----------:|:-------:|
|ふ|increment value|
|は|decrement value|
|え|increment pointer|
|ぇ|decrement pointer|
|!|output value|
|?|input value|
|「|loop start|
|」|loop end|
